# opt

Discrete Optimization package written in Go

## Problems

### bin
* `cover`       - Bin Cover
* `pack`        - Bin Packing

### interval
* `basic`       - Activity Selection
* `weight`      - Weighted Activity Selection

### knapsack
* `basic`       - 0-1 Knapsack
* `quad`        - Quadratic Knapsack

### sat
* `langford`    - Langford Pair
* `magicseries` - Magic Series
* `nqueens`     - N-Queens

### set 
* `cover`       - Set Cover 
* `pack`        - Set Packing
* `split`       - Set Splitting

### subseq
* `inc`     - Longest Increasing Subsequence
* `alt`     - Longest Alternating Subsequence

### subsetsum
* `basic`       - Subset Sum
